Introduction

This application is destined for users to report various problems in their city, in order for the authorities to resolve them. The application has two types of accounts: normal user and admin. The normal users can post different kinds of problems which can be marked as solved only by admins.

1. Login page

The first page in this application is the login page. Here, users can authenticate using their facebook account, or they can create a new account only for the application. Admins or authorities will make a request by mail, providing some proof that they are really authorities.

2. Register page

If the user does not have and account, he can create one by clicking the register tab, which will open a registration page where he will be asked to provide the data required for the account.

3. Posts

This is the page where all the posts will appear, added by users or posts from Facebook that have tagged the CitiZen page. Every post can be upvoted or downvoted, depending on the user's preference. The posts will be ordered by the number of upvotes. The user can select a post to view details about it, opening the Post Page, can add a post by clicking the Add post button and can search for some specific post in the website. By clicking the Advanced search, the user can select a specific category.

4. Post page

By clicking the Add post button, a user can submit a post with a problem he encountered. Other users can access the post page to comment, upvote or downvote it. Each post will have a status (unresolved / resolved). Authorities will have a Mark as resolved button to change the status of the problem in the post when they solve it.

5. Statistics

The statistics page will display the number of users that upvoted a post, if the authorities have worked on the problem, if it got resolved or not.
